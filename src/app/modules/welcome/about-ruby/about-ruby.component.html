O que é a linguagem ruby?

Ela é mais uma das opções de linguagens de programação disponíveis no mercado, usamos tais linguagens para programar distintos tipos de dispositivos eletrônicos, como por exemplo computadores, celulares, sensores. Portanto, é através das linguagens de programação que passamos instruções para tais dispositivos, conseguindo assim programá-los para executarem ações desejadas.

O grande diferencial de ruby para a maioria das outras alternativas é o fato dela ser uma linguagem feita para nós programadores! Programar em ruby é algo normalmente prazeroso e simples. Ela possui sintaxe elegante e recursos poderosos, fatores que proporcionando o aumento de produtividade de seus adeptos.

A melhor amiga do programador!

Foi criada pelo japônes Yukihiro Matsumoto (Matz) em 1993, escrita em c (linguagem de programação) e disponibilizada ao público em 1995. Seu criador se inspirou em outras linguagens de programação, como: Lisp, Perl, Smalltalk, Eiffel, e Ada.

Projetada para ser uma linguagem de programação produtiva e que carrega um conjunto de características, como: multiparadigma, interpretada,tipagem dinâmica, tipagem forte, portabilidade, legibilidade, redigibilidade etc.

Multiparadigma...

Paradigma nada mais é do que um estilo de programação.

Uma linguagem multiparadigma caracteriza-se por agrupar mais de um paradigma de programação para assim melhor atender as nossas necessidades, dessa forma, podemos trabalhar com vário estilos de programação usando uma única linguagem.

Ruby abrange de forma equilibrada os paradigmas de programação imperativa, funcional e orientado a objetos.

Imperativa: é um paradigma aliado a arquitetura de computadores, arquitetura que foi criada por von Neumann. Diversas linguagens de programação foram projetadas seguindo esse paradigma. Algumas catacterísticas das linguagens imperativas são o uso de variáveis, atribuição de valores e fluxo de execução sequencial de comandos.

Funcional: é um paradigma de programação baseado em chamadas de funções e que evita estados ou dados mutáveis. Prega pelo uso de constantes e não de variáveis, é declarativa e não imperativa. Atualmente, é um dos paradigmas de programação mais populares.

Orientado a objetos: é um paradigma de programação que visa representar o mundo natural dentro do mundo computacional. Algumas de suas características são o uso de classes, herança, polimorfismo, interfaces etc. Ruby é uma linguagem puramente orientada a objetos, tratando tudo como um objeto.

Interpretada...

Os dispositivos que programamos não entendem a nossa língua, eles conversam através de linguagem de baixo nível ou linguagem de máquina. Tal tipo de linguagem é complexa e de difícil entendimento para nós seres humanos, por isso utilizamos linguagens de programação de alto nível. Uma linguagem de alto nível possui semântica próxima a nossa, no entanto, os dispositivos não utilizam linguagem de alto nível.

Para que os dispositivos entendam as nossas instruções precisamos recorrer a um tradutor, este é programa que converte nossas instruções para linguagem de máquina. O tradutor de ruby é o que chamamos de interpretador.

Ser uma linguagem interpretada significa que suas instruções são traduzidas no momento de suas execuções, diferentemente de linguagens compiladas, onde primeiro o código é compilado e depois um arquivo executável é concebido.
