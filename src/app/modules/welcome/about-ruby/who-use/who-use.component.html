Quem usa ruby?

Ruby é uma linguagem de propósito geral, muito madura e provada no mercado, sendo utilizada para construção de diversas soluções tecnológicas. Sua área de maior atuação é no desenvolvimento de aplicações web. Algo muito influenciado pelo sucesso do super framework Ruby on Rails, ferramenta essa escrita em ruby. Outro framework ruby de destaque no mercado é o Sinatra.
