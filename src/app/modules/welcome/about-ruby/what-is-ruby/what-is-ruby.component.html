O que é a linguagem ruby?

Ela é mais uma das opções de linguagens de programação disponíveis no mercado, usamos tais linguagens para programar distintos tipos de dispositivos eletrônicos, como por exemplo computadores, celulares, sensores. Portanto, é através das linguagens de programação que passamos instruções para tais dispositivos, conseguindo assim programáramos para executarem ações desejadas.

O grande diferencial de ruby para a maioria das outras alternativas é o fato dela ser uma linguagem feita para nós programadores! Programar em ruby é algo normalmente prazeroso e simples. Ela possui sintaxe elegante e recursos poderosos, fatores que proporcionando o aumento de produtividade de seus adeptos.

A melhor amiga do programador!

Foi criada pelo japônes Yukihiro Matsumoto (Matz) em 1993, escrita em c (linguagem de programação) e disponibilizada ao público em 1995. Seu criador se inspirou em outras linguagens de programação, como: Lisp, Perl, Smalltalk, Eiffel, e Ada.

Nos primeiros anos de vida de ruby ela ficou praticamente restrita ao país de sua origem e só por volta dos anos 2000 chegou aos EUA (Estados Unidos da América). O marco principal dessa introdução do ruby ao EUA ocorreu em setembro de 2000, quando o primeiro livro em inglês, Programming Ruby, foi publicado. Então ruby começou a fazer sucesso e como o inglês é a língua universal conseguiu chegar a outras nações.

Em 2005 nasce o revolucionário Ruby on Rails (Rails ou RoR), o super framework para o desenvolvimento de aplicações web escritas em ruby. O próprio Ruby on Rails foi escrito em ruby. Provavelmente você se interessou em ruby porque ouviu falar sobre a Rails, ele realmente é uma ferramenta diferenciada que consegue aumentar a produtividade de seus adeptos e deixar o processo de desenvolvimento de aplicações mais divertido. Mas o foco aqui será a linguagem ruby. Depois que tiver uma boa base sobre ruby terá enorme facilidade em trabalhar com o Rails.

O rails atraiu muitos olhares e como consequência o ruby também. A partir desse período a linguagem se popularizou e ganhou destaque no cenário mundial.

Ela foi projetada para ser uma linguagem de programação produtiva e para carregar um conjunto de características, como: multiparadigma, interpretada,tipagem dinâmica, tipagem forte, portabilidade, legibilidade, redigibilidade etc.
