Aprenda a programar em ruby!

Se você nos encontrou é porque se interessou por ruby, uma das melhores linguagens de programação do mercado. Certamente, está procurando uma fonte de conteúdos onde possa aprender sobre essa linguagem tão poderosa e fascinante. Portanto, seja bem-vindo(a) ao projeto Aux-Ruby-Doc!.

O Aux-Ruby-Doc (ARD) é uma plataforma que visa ajudar no processo de aprendizagem da linguagem de programação ruby. Todo conteúdo compartilhado aqui foi inserido por quem já esteve na mesma situação que você, com muita curiosidade e ao mesmo tempo um pouco perdido. Sabemos que quando iniciamos os estudos sobre uma nova tecnologia normalmente encontramos algumas dificuldades, por isso projetamos o ARD, para ajudar pessoas aprenderem a programar em ruby.

Por que ruby não foi utilizado?

Você deve está se perguntando porque o ruby não foi empregado no desenvolvimento do Aux-Ruby-Doc. Decidimos não usar ruby para o desenvolvimento do projeto, pois a plataforma na verdade nada mais é do que um site de compartilhamento de conteúdo, dados não são gravados, autenticação e gerenciamento de permissões não são realizados, ou seja, se trata apenas de uma aplicação front-end.

Portanto, chegamos a conclusão de que usar ruby meramente para construção de front-end seria fazer mau uso da tecnologia. Não estamos falando que ruby não possa ser usado para desenvolver um projeto web de ponta a ponta, participando do back-end e front-end, existem situações que isso se faz necessário e o sucesso do Ruby on Rails mostra isso.

No entanto, nossa plataforma não possui um back-end, apenas trabalha com html. Defendemos que ruby deva ser usado para o processamamento e manipulação de dados, atuando principalmente como back-end de aplicações e não se preocupando UNICAMENTE em renderizar html. Como não era a melhor escolha optamos por um framework estruturado para a construção de uma aplicação de página única (Single Page Application ou SPA), escolhendo o Angular para tal missão.
