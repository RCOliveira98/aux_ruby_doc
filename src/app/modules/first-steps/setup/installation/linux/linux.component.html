Instalação do RVM

O RVM é uma ferramenta de linha de comando que nos possibilita instalar, gerenciar e trabalhar com vários ambientes ruby de maneira muito simples e fácil. Tal ferramenta permite por exemplo instalarmos mais de uma versão da linguagem ruby. Recomendamos o seu uso.

01) Instale GPG keys: usada para verificar o pacote de instalação

gpg2 --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3 7D2BAF1CF37B13E2069D6956105BD0E739499BDB

02) Instale o CURL: é uma ferramenta de linha de comando para manipulação de URLs e transferência de dados

sudo apt-get install curl

03) Instale RVM

\curl -sSL https://get.rvm.io | bash -s stable

04) Após a instalação do rvm ele indicará a necessidade de rodar um comando para começar a utilizar o rvm. Analise seu terminal ele deverá mostrar um comando no formado do script abaixo:

source /home/SEU_USER/.rvm/script/rvm

05) Liste as opções de instalação ruby

rvm list known

06) Instale a versão ruby de sua preferência

rvm install 2.7.0

# 2.7.0 é a versão
