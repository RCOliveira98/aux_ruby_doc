Olá, mundo!

Dentro do universo da programação existe uma tradição de que o primeiro programa feito em uma determinada linguagem deve ser o 'Olá, mundo!'. Tradição criada por que segundo a lenda aqueles que não a seguem jamais serão bons programadores.

Agora surge a dúvida: como criar um programa para escrever uma simples mensagem na tela ou console de um dispositivo? Para isso precisamos pedir ao dispositivo para escrever algo através de alguns comandos, dentre eles o puts.

puts 'Olá, mundo!'

Pronto, como uma única instrução você foi capaz de escrever seu primeiro programa em ruby. Muito fácil, né? Agora, como essa 'mágica' ocoreu? O responsável pelo sucesso do seu programa foi o método puts. Chamadas a métodos em ruby não precisam de parêteses '()', é algo opcional, diferentemente de outras linguagens como java, c e c++. Sendo uma característica da linguagem que é conhecida como syntactic sugar (áçucar sintático), deixando o código mais limpo, ou seja, menos poluído.
