Olá, mundo!

Dentro do universo da programação existe uma tradição de que o primeiro programa feito em uma determinada linguagem deve ser o 'Olá, mundo!'. Tradição criada por que segundo a lenda aqueles que não a seguem jamais serão bons programadores.

Agora surge a dúvida: como criar um programa para escrever uma simples mensagem na tela ou console de um dispositivo? Para isso precisamos pedir ao dispositivo para escrever algo através de alguns comandos, dentre eles o puts.

puts 'Olá, mundo!'
