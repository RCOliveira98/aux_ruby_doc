Variáveis

A maioria dos programas desenvolvidos precisa armazenar dados, como por exemplo: data de nascimento, endereço, número de cartão de crédito ou débito, nome etc. Na computação existe o que é denominado de variável, que em poucas palavras é uma posição de memória do computador destinada a armazenar valores de um dado tipo de dado. Para manipular essas variáveis (endereços de memória) são atribuídas a elas nomes.

Declaração e atribuição de valores a variáveis

Para delcarar uma variável em ruby precisamos definir apenas um nome e atribuir um valor para ela. Confira o exemplo abaixo:

message = 'Seja bem-vindo(a) ao mundo ruby!'
puts message

Uau! Fizemos a delcaração de uma variável chamada de message e pedimos para ela armazenar o texto 'Seja bem-vindo(a) ao mundo ruby!'. Por fim utilizamos o método puts para o dispositivo imprimir o conteúdo de nossa variável.

Muito bem, agora perceba o quanto código ruby é próximo de nossa linguagem humana. Pois a leitura da instrução: message = 'Seja bem-vindo(a) ao mundo ruby!' pode ser facilmente interpretada por nós como: variável message receba (armazene) o valor de 'Seja bem-vindo(a) ao mundo ruby!'

Perceba que para informarmos que a variável message armazenaria o conteúdo 'Seja bem-vindo(a) ao mundo ruby!' fizemos o uso do símbolo de igual (=), que no mundo da programação é conhecido como operador de atribuição.

Uma das coisas interessantes em ruby é que variáveis não precisam serem declaradas previamente antes de seu uso, assim como o tipo de dado não precisa ser informado, pois ela é inteligente o suficiente para descobrir o tipo de dado com base no valor armazenado, característica chamada de inferência de tipos.

Muito bom, mas que tal agora o usuário do programa definir o conteúdo de uma determinada variável?

puts 'Informe seu nome:'
nome = gets
puts nome

O que fizemos? Informamos ao computador para imprimir a mensagem informe seu nome:, depois para fazer a leitura do dado informado pelo usuário, pegar o que foi capturado e armazenar na variável nome, por fim imprimir o conteúdo da variável nome.

A captura do valor informado pelo usuário só foi possível por causa do método gets. Tal método lê tudo que encontrar até uma mudança de linha provocada por exemplo pela tecla enter. Ao término da leitura o conteúdo lido é retornado, no exemplo anterior o retorno do método foi atribuído a variável nome.

Retorno do método gets: por padrão o método sempre retorna uma string (um tipo de dado), não importando o que seja digitado. Pois uma conversão automática para o tipo string é realizada. No próximo tópico será abordado os tipos de dados da linguagem.

Uso de aspas (''): quando se deseja que variáveis recebam valores que sejam caracteres (letras, palavras , textos etc) tais valores devem ficar entre aspas, sejam elas simples ou compostas:
texto = 'Aspas simples' ou texto = "Aspas duplas"
