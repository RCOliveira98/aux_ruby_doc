Variáveis

A maioria dos programas desenvolvidos precisa armazenar dados, como por exemplo: data de nascimento, endereço, número de cartão de crédito ou débito, nome etc. Na computação existe o que é denominado de variável, que em poucas palavras é uma posição de memória do computador destinada a armazenar valores de um dado tipo de dado. Para manipular essas variáveis (endereços de memória) são atribuídas a elas nomes.

puts 'Informe seu nome:'
nome = gets
puts nome

O que fizemos? Informamos ao computador para imprimir a mensagem informe seu nome:, depois para fazer a leitura do dado informado pelo usuário, depois para pegar o que foi capturado e armazenar na variável nome, por fim imprimir o conteúdo da variável nome.

No exemplo acima, foi possível armazenar o nome do usuário em uma variável denominada de nome. Perceba que em ruby variáveis não precisam serem declaradas previamente antes de seu uso, assim como o tipo de dado não precisa ser informado, pois ela é inteligente o suficiente para descobrir o tipo de dado com base no valor armazenado, característica chamada de inferência de tipos.

A captura do valor informado pelo usuário só foi possível por causa do método gets. Tal método lê tudo que encontrar até uma mudança de linha provocada por exemplo pela tecla enter. Ao término da leitura o conteúdo lido é retornado, no exemplo anterior o retorno do método foi atribuído a variável nome.

Retorno do método gets: por padrão o método sempre retorna uma string (um tipo de dado), não importando o que seja digitado. Pois uma conversão automática para o tipo string é realizada. No próximo tópico será abordado os tipos de dados da linguagem.
