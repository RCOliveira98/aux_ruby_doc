Interpolação

Interpolação de strings (string interpolation) é o processo de concatenar duas ou mais strings. Imagine que precise retornar uma mensagem de boas vindas com o nome do usuário de seu programa. A pergunta que fica é como isso seria feito? A resposta é simples, através da concatenação! String interpolation é a maneira mais convencional e elegante de concatenar valores resultando em uma string única. Um detalhe bastante importante é que esse recurso deve ser realizado somente entre aspas duplas ("").
