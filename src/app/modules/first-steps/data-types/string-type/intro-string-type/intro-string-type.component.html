String

Durante a rotina de um programador é muito comum a necessidade de armazenar textos ou caracteres, como: nome de usuário, senha, endereço, e-mail etc. Na computação chamamos esses dados de tipo string! Em ruby qualquer valor que for declarado entre aspas será tratado como do tipo string.

email = 'ard@email.com'
puts email
password = "123ruby"
puts password

Perceba que variáveis do tipo string podem ser declaradas com aspas simples ('') ou duplas (""). A delcaração por meio de aspas simples chega a ser um pouco mais perfomática pois a verificação da presença de interpolação -- em breve falarei sobre isso -- não é feita. Mas a alternativa a declaração de variáveis com aspas duplas tem sua finalidade. Análise o código abaixo:

district = 'Caixa D'água'
puts district

O erro ocorreu porque confundimos o interpretador de código ruby com o uso de aspas simples dentro do texto. Não podemos declarar string com aspas simples e depois fazer o uso de aspas simples dentro do próprio texto! Por que? Porque o interpretador não vai conseguir encontrar o fim da string. A alternativa a esse problema é recorrer a declaração de strings com aspas duplas.

district = "Caixa D'água"
puts district
