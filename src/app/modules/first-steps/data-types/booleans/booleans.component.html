Tipos TrueClass e FalseClass

Tipos de dados que representam os famosos valores booleanos: true (verdadeiro) e false (falso).

value = true
puts value.class # resultado: TrueClass
value = false
puts value.class # resultado: FalseClass

Na maioria das linguagens de programação são representados pela class Boolean ou Bool, mas ruby preferiu ter uma classe para cada um dos valores.
