Ruby descobre o tipo de dado da variável

Você já descobriu o quão simples é declarar variáveis em ruby. Precisamos apenas associar um nome a um determinado valor e o ruby cuida do restante do trabalho para você.

Ruby descobre o tipo de dado da variável e separa um pedaço de memória do seu dispositivo (computador, tablet etc) que seja suficiente -- de acordo com o tipo de dado -- para armazenar o valor desejado, esse processo é conhecido como alocação de memória, mas não precisa se preocupar com isso pois o ruby cuida de todo o processo de gerencimanento de memória por você.

Contudo, se você já conhece outras linguagens como Java e C acredito que tenha em algum momento se perguntado por que não precisamos declarar os tipos de dados das variáveis. O ruby é muito inteligente, com base no valor armazenado na variável ele determina o tipo de dado, característica chamada de inferência de tipos.

user_name = 'Fulano123'
code = 1020
is_man = true

A declaração de 03 (três) variáveis foi feita, mas você pode está se perguntando quais são os tipos de dados das variáveis declaradas? Como podemos descobrir isso? Muito simples, basta perguntar para a própria variável! Existe um método chamado class que será útil nessa missão, analise o código abaixo:

user_name = 'Fulano123'
puts user_name.class # resultado: String
code = 1020
puts code.class # resultado: Integer
is_man = true
puts is_man.class # resultado: TrueClass
