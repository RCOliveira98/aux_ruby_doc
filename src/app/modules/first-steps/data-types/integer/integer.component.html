Tipo Integer

Um tipo de dado que representa os números inteiros. Alguns exemplos de valores do tipo Integer são: 123, -31, 4223432, -123242, 0, 4142 etc.

number = 90
puts number # resultado: 90
puts number.class # resultado: Integer
Lendo valores do tipo Integer

Agora vamos coletar um valor do tipo Integer informado pelo usuário do programa.

puts "Informe sua idade: "
age = gets
puts number.class # resultado: String

Ops... Algo deu errado? Ao informar um valor inteiro imaginava-se que a variável age estivesse armazenando um valor do tipo Integer e não do tipo String como revelado pelo programa. Não sei se recorda mas já foi dito aqui que por padrão o retorno do método gets é um valor do tipo String, pois o método captura o valor digitado e converte automaticamente para String.

Portanto, precisamos converter o valor retornado pelo método gets para Integer, já que o mesmo é um número inteiro que foi convertido para String automaticamente pelo gets. Para isso basta utilizar o método to_i, observe:

puts "Informe sua idade: "
age = gets.to_i
puts number.class # resultado: Integer
