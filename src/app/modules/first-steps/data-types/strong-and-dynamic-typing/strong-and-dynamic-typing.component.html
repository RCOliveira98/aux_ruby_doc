Sistema de tipagem

Você já aprendeu a declarar variáveis para armazenar dados de interesse de seus programas. Também já sabe que o valor de uma variável pode ser alterado. O que você ainda não viu é que uma variável pode armazenar valores de tipos difentes de dados. Além do detalhe que ruby possui tipagem forte.

Tipagem dinâmica

Na maioria das linguagens tradicionais uma variável só pode receber dados de um único tipo, tipo esse definido no momento de sua declaração. Tal característica é denominada de tipagem estática.

Mas a linguagem ruby entende que uma mesma variável não só pode armazenar diferentes valores, mas que tais podem ser de diferentes tipos. Ou seja, uma variável pode armazenar em um momento um texto qualquer e depois um valor numérico. Chamamos essa característica de tipagem dinâmica.

Observer o código abaixo onde uma variável chamada de auxiliary será declarada, a mesma receberá valores de tipos diferentes. Ao final de cada atribuição pediremos para o tipo de dado que a variável carregará seja exibido.

auxiliary = 'Romário Coelho'
puts auxiliary.class # resultado: String
auxiliary = 1998
puts auxiliary.class # resultado: Integer

Parece ser algo muito simples e que não merece atenção, mas muitas linguagens adotam tipagem estática e que como consequência não permitem a execução do exemplo acima.
