Tipos de dados

Os dados que são manipulados por programas possuem tipos. Por exemplo, o nome de um cliente não é do mesmo tipo do valor total comprado por ele em uma loja.

Os tipos de dados possuem muita importância dentro do mundo da programação. Eles definem quais operações poderão serem realizadas com os dados. Acredito que concorde que seria muito estranho pedir para o computador dividir a letra A pela letra C.
