RubyGem

Se você vêm do javascript conhece os famosos gerenciadores de pacotes npm e yarn, no ruby temos o rubyGem ou simplesmente gem. Através dele você poderá ter acesso a diversas gems (bibliotecas e programas) que poderão lhe auxiliar em distintas situações, como exemplos de recursos que poderá obter por meio do gerenciador de pacotes temos o irb e o pry.

Atenção, no momento que o ruby é instalado ele já carrega o seu gerenciador de pacotes!

Instale o irb:

gem install irb

Instale o pry:

gem install pry

Agora para usar basta digitar em seu terminal o comando pry ou irb de acordo com o qual foi instalado em sua máquina. Depois pressione a tecla ENTER.
