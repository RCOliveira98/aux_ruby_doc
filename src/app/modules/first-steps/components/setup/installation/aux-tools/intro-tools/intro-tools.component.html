Para que possa testar o poder do ruby um simples editor de texto instalado em seu computador será o suficiemte. Pois com a linguagem ruby instalada bastará criar um arquivo e salva-lo com a extensão .rb. Depois execute o arquivo através do comando: ruby nome_arquivo.rb

Dessa maneira poderá testar os exemplos aqui disponibilizados sem problemas. No entanto, os primeiros exercícios são extremamentes simples e por isso recomendamos a instalação de utilitários que lhe permitirão exercitar a linguagem ruby sem a necessidade de salvar arquivos.

Existem duas ótimas opções de shell interativo, o IRB e o PRY. Através deles poderá executar comandos ruby diretamente no terminal (console) e assim praticar sem ter que armazenar arquivos em sua máquina.

  • IRB significa "Ruby interativo" e é uma ferramenta para executar código ruby de forma interativa. O comando irb do seu shell iniciará o interpretador.
  • PRY é um console para desenvolvedor. Sendo uma alternativa ao IRB com poderosos recursos de introspecção.

Você pode escolher o de sua preferência, mas particulamente temos uma pequena preferência ao PRY.
