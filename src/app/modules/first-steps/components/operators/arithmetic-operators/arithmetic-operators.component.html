Prática

Agora chegou a hora de testar os operadores aritméticos no console (pry ou irb), confira:

num1 = 10
num2 = 3
puts num1 + num2 # resultado = 13
puts num1 - num2 # resultado = 7
puts num1 * num2 # resultado = 30
puts num1 / num2 # resultado = 3.33
puts num1 % num2 # resultado = 1
puts num1 ** num2 # resultado = 1000
