Tipo Float

Um tipo de dado que representa os números de ponto flutuantes ou fracionários. Alguns exemplos de valores do tipo Float são: 34.90, -23.45, 93284.432, -2343.43, 24.12, 5343.91 etc. Detalhe, use sempre o . (ponto) ao invés da . (vírgula);

number = 90.52
puts number # resultado: 90.52
puts number.class # resultado: Float
Lendo valores do tipo Float

Agora vamos coletar um valor do tipo Float informado pelo usuário do programa.

puts "Informe seu salário: "
salary = gets
puts salary.class # resultado: String

Ao informar um valor de ponto flutuante imaginava-se que a variável salary estivesse armazenando um valor do tipo Float e não do tipo String como revelado pelo programa. Lembre-se que por padrão o retorno do método gets é um valor do tipo String, pois o método captura o valor digitado e converte automaticamente para String.

Portanto, precisamos converter o valor retornado pelo método gets para Float, já que o mesmo é um número inteiro que foi convertido para String automaticamente pelo gets. Para isso basta utilizar o método to_f, observe:

puts "Informe seu salário: "
salary = gets.to_f
puts salary.class # resultado: Float
